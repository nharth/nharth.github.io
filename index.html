Analysis of Players in the 2022/2023 Premier League¶

Noah Nazrul Hisham¶

The English Premier League (EPL) is the highest division of professional soccer in England. The league consists of 20 teams. Each team in the league may have different objectives. Some want to achieve first place, while others may simply want to avoid relegation. Relegation occurs at the end of the season, where the 3 teams with lowest amount of points are sent to play in the second division during the next season. Relegation can have serious financial consequences. On the other side of the table, teams fight to win the league by obtaining the most points. Additionally, some teams just aim to get in the top four positions.Getting into the top 4 ensures Champions League qualification. The Champions League where the best teams in Europe get to play against each other in order to see who is the greatest. It is arguably one of the greatest competitions in all sports.

As said previously, the English Premier League has 20 teams.Currently, there are a total of 557 players in the English Premier League. In this project, player statistics will be analyzed in order to walk the reader though the data science pipeline.The data science pipeline for this tutorial includes data curation, parsing, and management; exploratory data analysis; hypothesis testing and machine learning for analysis; and anything learned through the tutorial.

There are several kinds of statistics that are measured for every player. Some that come to mind are goals, assists, games played, etc. This tutorial will introduce the reader to several other statistics that they may not have heard of. These statistics may be as variables used in machine learning models to predict other variables. In this tutorial, a multiple linear regression model will be used to find the relationship between multiple independent variables and one dependent variable. The dependent variable will be the salary of the player. Ultimately, we'd like to find out how to predict the salary of a player based on their performance.

Data Collection¶

The first section in this tutorial is to collect data. But before that, we need to import the libraries needed for the tutorial. These libraries are pandas, matplotlib, seaborn, and sklearn.

The first data set contains statistics of English Premier League players. The columns include things like name, age, goals scored, assists, team, etc. It can be found here: "https://fbref.com/en/share/NsKcV". We will assign that link to a variable and parse the html using pandas.read_html in order to get a DataFrame that contains all the data.

Cleaning the data¶

So in order to clean the data, we will create another DataFrame that will only have single columns. The reason we are doing this is because we want to merge this DataFrame with another later on. Also, in this tep, we will modify the nation column so that the nation is represented by a 3 letter code.

import re

def get_usd(x):
    match = re.search(r'\$([\d,]+)', x)
    if match:
        return float(match.group(1).replace(',', ''))
    else:
        return None
